# Python-Programming-Lab-2025-26
This repository is created to support students of the Python Programming course (MMC101) for MCA 1st Semester at Gogte Institute of Technology. It includes the official list of lab programs, complete Python source codes, and a sample journal writing format to guide students throughout the academic year.

---

##  Lab Programs
The lab programs cover fundamental and core Python concepts such as:
- Decision Control Structure Statements, List/Dictionary
- User Defined Functions
- Working with database
- Object Oriented Concepts
- Overloading Operator
- GUI Programming
- Numpy/Pandas
- Visualization

(Refer to the **Program_List** folder for the complete list.)

---

## Journal Writing Guidelines
The **Journal_Format** folder contains:
- **sample lab journal write-up**
- Standard format followed for:
  - Title
  - Objectives
  - About the Experiment (Concepts used in the program)
  - Source code
  - Output (Printout to be attached)
  - Learning conclusion
  - Course Outcome Attained

Students are advised to follow this format while maintaining their lab journals.

---

## How to Use This Repository
1. Refer to the **Program List** before attending the lab.
2. Use the **Source Codes** to understand program logic.
3. Practice writing and executing the programs on your own.
4. Follow the **Journal Format** while documenting experiments.
5. Use this repository strictly for **learning and reference purposes**.

---

## Academic Integrity
- Students are encouraged to **understand and practice** the programs.
- Blind copying of code is **not recommended**.
- Modify and experiment with the programs to enhance learning.
