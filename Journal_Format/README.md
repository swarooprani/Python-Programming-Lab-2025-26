# Journal Writing Format

This folder contains the sample laboratory journal writing format
for the Python Programming Lab (MMC101).

Students are expected to follow this format while maintaining
their lab records.

## Standard Sections in Journal
Each experiment should include:
1. Experiment No. with Title
2. Objectives
3. Source Code
4. Output (Printout of the result after the program execution)
5. Learning Outcomes
6. conclusion

A sample journal write-up is provided in this folder for reference.
