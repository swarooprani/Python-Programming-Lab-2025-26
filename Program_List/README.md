## List of Lab Programs

### Program 01: Dictionary Management System
Write a Python program that manages a dictionary of words and their meanings with the following operations:
1. Add a new word and its meaning  
2. Find the meaning of a word  
3. Display all words and meanings  
4. Delete a word  
5. Modify the meaning of an existing word  
6. Quit the program  

---

### Program 02: Menu Driven Number Operations Using Functions
Write a menu-driven Python program using **user-defined functions** to perform the following operations:
1. Check whether a number is prime  
2. Find the factorial of a number  
3. Check whether a number is even or odd  
4. Check whether a number is a perfect number  
5. Exit  

---

### Program 03: Database Operations – Census Data
Create a Python program to perform database operations using SQLite:
1. Create a database named `census.db`  
2. Create a table `Density` with district name, population, and land area  
3. Insert data into the table (assume data)  
4. Display the contents of the table  
5. Display populations  
6. Display districts with population less than 1 million  
7. Display districts with population less than 1 million or greater than 5 million  
8. Display districts that do not satisfy the above condition  
9. Display districts with land area greater than 200,000 sq. km  
10. Display population density of each district  

---

### Program 04: Bank Account System (Object Oriented Programming)
Develop a Python program using OOP concepts to simulate a bank account system with the following features:
1. Create Account  
2. Deposit Money  
3. Withdraw Money  
4. Check Balance  
5. Display Account Details  
6. Exit  

The program should be menu-driven and continue until the user chooses to exit.

---

### Program 05: Operator Overloading – Distance Class
Write a Python program to demonstrate **operator overloading** by overloading the `+` operator to add two objects of a class `Distance`.

---

### Program 06: GUI Billing Application using Tkinter
Create a GUI application using Tkinter to compute the final billing amount with the following conditions:
- Price of Product 1 = ₹500 per unit  
- Price of Product 2 = ₹50 per unit up to 50 units, otherwise ₹45  
- Price of Product 3 = ₹100 per unit (minimum quantity: 10 units)  
- Display the final bill amount when the **Compute Bill** button is clicked  

---

### Program 07: NumPy and Pandas Operations
Write a Python program using **NumPy and Pandas** to perform the following:
- Create an identity matrix  
- Find the square root of each element in an array  
- Sort an array  
- Square each element in an array  
- Find logarithm of each element  
- Create arrays of zeros and ones  
- Find the mean of an array  
- Create and merge two DataFrames  

---

### Program 08: Data Visualization using Matplotlib
Create the following plots using **Matplotlib**:
- Line Plot  
- Histogram  
- Bar Chart  
- Scatter Plot  
- Pie Chart
